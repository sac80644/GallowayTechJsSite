Typescript - General Tips and Tricks

I started using Typescript somewhere mid 2018, but have been using it daily for the last few months. I sometimes feel like I am in an odd limbo between C# and Javascript. Using Typescript is an easier transition for someone who is used to a strongly typed object oriented language such as C# than going straight to Javascript. But I don't need an easy transition and I really wanted to do more pure Javascript. Nevertheless, I do like it and it is my day job for now. Here are some tidbits of info. And because Typescript is a superset of Javascript, I can't help but mix them together.

Spread Operator

I came across something like this not too long ago:

var foo = new Set(...args);
And I thought, what the heck is that?!? So I looked it up. I found the ... is referred to as the spread operator, or on the Mozilla site as Spread Syntax. This looks exactly like Rest Syntax but not the same. Anyways, today I used it to push an array of object into another array of object like such:

let firstArray: MyObject[] = [{key: "value1"}, {key: "value2"}, {key: "value3"}];
let secondArray: MyObject[] = [{key: "value4"}, {key: "value5"}, {key: "value6"}];
firstArray.push(...secondArray);
And voila. The secondArray's objects are expanded out and pushed into the firstArray. This can also be done with Array.concat() or some other methods but I like this one...
