How to Excel in What You Do

Here is a simple list of very straight forward things anyone can do to set yourself apart from other workers. Knowing and excelling at your craft is definately the name of the game, but I have found that there are just basic skills that so many talented developers, and other professions too, lack.

Seven Daily Habits of Successful Professionals

  1. Show up to work on time
  2. Put in a full day's work
  3. Keep sick days / appointments to a minimum and make sure to be honest and communicate early
  4. Strive to do your best and give your best
  5. Take ownership of your projects
  6. Be proactive
  7. SOLID

Why isn't this documented?

Ok, so some of these things are documented, but perhaps hard to find. Just quick thoughts here, things I keep forgetting so I'm writing them down, etc.

Git Hints

Undo working copy modifications of one file in Git?
git checkout -- file
git checkout v1.2.3 -- file         # tag v1.2.3
git checkout stable -- file         # stable branch
git checkout origin/master -- file  # upstream master
git checkout HEAD -- file           # the version from the most recent commit
git checkout HEAD^ -- file          # the version before the most recent commit
